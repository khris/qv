qv
==


Overview
--------

__qv__ is a simple puzzle game made with _[Qt Quick][1]_. For now, _it is under
construction_. __qv__ will be support _MS Windows_, _Linux_, _Android_ and
maybe _Symbian_.

[This][2] is demo video.
